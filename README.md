# Universidad API

API REST para gestión de estudiantes universitarios.

## Documentación

La documentación completa está disponible en [Writerside/topics/README.md](Writerside/topics/README.md)

## Tecnologías

- Java 21
- Spring Boot 3.4.4
- Maven

## Instalación

```bash
mvn clean install
```

## Ejecución

```bash
mvn spring-boot:run
```
